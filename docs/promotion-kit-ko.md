## 초기 홍보 순서

1. 도쿄 여권 분실, 방콕 휴대폰 분실, 다낭 병원 가이드부터 개별 게시합니다.
2. 각 게시물은 메인 화면이 아닌 해당 가이드 URL로 직접 연결합니다.
3. 동일 문구를 반복하지 말고 실제 체크리스트 3~5개를 요약해 게시합니다.
4. 여행 카페와 커뮤니티에서는 홍보글보다 질문에 답하면서 관련 가이드를 맥락 있게 공유합니다.
5. 매주 Search Console에서 노출은 있으나 클릭률이 낮은 검색어의 제목과 설명을 개선합니다.

## 측정 기준

- GA4 유입 채널과 인기 가이드
- Search Console 검색어별 노출·클릭률·평균 순위
- 공유 버튼 사용 이후 직접·소셜 유입 변화
- 상위 20개 가이드의 28일 단위 성장률
